# Markdown Fundamentals

This is a **basic markdown** post without Liquid syntax to test the parser's handling of different content types.

## Basic Syntax

### Headers
Use `#` for headers, from H1 to H6.

### Text Formatting
- **Bold text**
- *Italic text*
- `Inline code`
- ~~Strikethrough~~

### Lists
1. Ordered list item
2. Another item
   - Nested unordered item
   - Another nested item

### Links and Images
[Link to example](https://example.com)

### Code Blocks
```javascript
function hello() {
  console.log("Hello, world!");
}
```

### Blockquotes
> This is a blockquote.
> It can span multiple lines.

### Tables
| Header 1 | Header 2 |
|----------|----------|
| Cell 1   | Cell 2   |
| Cell 3   | Cell 4   |
